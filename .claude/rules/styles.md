## Style Rules

- Use native CSS nesting, no preprocessors
- Try to reuse components, such as buttons, links, inputs, or selects
- Try to minimize styles duplicating
- Try to minimize usage of such styles as font-family, font-size, and font-weight. If
  fonts are the same as a global styles, you can skip adding font styles for such elements
- if color uses one or two time on website, don't create a variable for it. but if it is
  three or more time create a variable and replace hardcoded colors
